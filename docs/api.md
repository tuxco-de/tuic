# TUIC RESTful API 接口文档

TUIC 服务端提供了一套基于 HTTP 的 RESTful API，用于遥测、查询在线状态和管理连接。所有 API 响应统一采用 JSON 格式。

## 认证方式

如果 `config.toml` 中的 `[restful]` 模块配置了 `secret`，则调用所有 API 接口时，必须在请求头中携带 Bearer Token：

```http
Authorization: Bearer <secret>
```

若 `secret` 为空，或未提供 `restful.secret`，则可直接调用接口。

---

## 接口详情

### 1. 获取在线用户统计
- **接口**：`GET /online`
- **能力**：获取当前所有在线用户的连接数统计。
- **返回**：一个以用户 UUID 为键，值为该用户并发连接总数的 JSON 对象。
- **返回示例**：
  ```json
  {
    "f0e12827-fe60-458c-8269-a05ccb0ff8da": 5,
    "1a2b3c4d-5e6f-7g8h-9i0j-k1l2m3n4o5p6": 1
  }
  ```

### 2. 获取详细客户端来源
- **接口**：`GET /detailed_online`
- **能力**：获取更详细的在线客户端信息。
- **返回**：列出每个 UUID 当前所有连接客户端的真实来源 IP 和端口号的 JSON 对象。
- **返回示例**：
  ```json
  {
    "f0e12827-fe60-458c-8269-a05ccb0ff8da": [
      "203.0.113.10:12345",
      "203.0.113.10:12346"
    ]
  }
  ```

### 3. 获取流量统计
- **接口**：`GET /traffic`
- **能力**：获取自服务端启动（或上次重置）以来的用户级总流量统计。**注：流量统计信息仅保存在内存中，重启后清空。**
- **返回**：包含各个 UUID 发送 (`tx`) 和接收 (`rx`) 字节数的 JSON 对象。
- **返回示例**：
  ```json
  {
    "f0e12827-fe60-458c-8269-a05ccb0ff8da": {
      "tx": 1048576,
      "rx": 2097152
    }
  }
  ```

### 4. 获取并重置流量统计
- **接口**：`GET /reset_traffic`
- **能力**：获取当前的流量统计数据，并在返回后**立即将所有用户的计数器清零**。适用于与定时任务系统（如 crontab）结合用于按时段计费。
- **返回**：与 `/traffic` 格式相同，返回清零前的数据快照。

### 5. 强制断开指定用户连接
- **接口**：`POST /kick`
- **能力**：强制踢出（断开）指定用户当前的所有活跃连接。
- **参数**：请求体（Body）需为包含想要踢出的 UUID 的 JSON 数组。
  ```json
  [
    "f0e12827-fe60-458c-8269-a05ccb0ff8da"
  ]
  ```
- **请求头**：需要包含 `Content-Type: application/json`。
- **行为**：查找到对应用户的所有在线会话，并以 `6002` 错误码（"Client got kicked"）主动关闭 QUIC 连接。
- **返回**：无正文（200 OK）。
